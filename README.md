# Quiz Application

This is a simple Quiz Application built using Spring Boot, MySQL, HTML, CSS, and JavaScript. The application includes features such as a registration page, login page, and quiz section where users can answer questions and submit their results.

## Features

- **Registration Page**: Users can register to create an account.
- **Login Page**: Registered users can log in to start the quiz.
- **Quiz Page**: Users can attempt 20 questions of 6 different types.
- **Result Page**: After submitting the answers, users can view their results.
- **Navbar**: A consistent navigation bar across all pages for easy access to Registration, Login, and Quiz sections.
